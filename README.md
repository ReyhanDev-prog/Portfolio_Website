## ✨ Fitur

- **Boot Sequence** — Animasi loading awal dengan progress bar (bisa di-skip)
- **Typing Animation** — Efek teks mengetik pada role (GAME DEVELOPER / DATA SCIENCE STUDENT / CREATIVE DEVELOPER)
- **Custom Cursor** — Cursor futuristik dengan lingkaran yang mengikuti (desktop only)
- **Scan Line** — Garis scanning pada frame profil
- **Progress Bar Animasi** — Skill bar dan EXP bar terisi saat section terlihat (Intersection Observer)
- **Project Cards** — Card dengan efek hover scanning dan border glow
- **Skill Database** — Node skill dengan efek hover "TECHNOLOGY IDENTIFIED"
- **Experience Timeline** — Timeline vertikal dengan node
- **Achievement Badges** — Badge unlocked/locked seperti dalam game
- **Contact Terminal** — Form dengan simulasi transmisi (ENCRYPTING → SENDING → SUCCESS)
- **Parallax Ringan** — Background dan avatar bergerak mengikuti posisi mouse
- **SFX Toggle** — Efek suara menggunakan Web Audio API (off by default)
- **Responsive Design** — Optimal untuk desktop, laptop, tablet, dan smartphone
- **Accessibility** — Semantic HTML, ARIA labels, keyboard navigation, focus states
- **Reduced Motion** — Menghormati pengaturan `prefers-reduced-motion`

## 🎨 Konsep Visual

- Background hitam kebiruan dengan grid digital transparan
- Warna utama cyan / electric blue
- Glow tipis dan elegan
- Frame HUD dengan sudut terpotong (clip-path)
- Dekorasi sirkuit minimal
- Bukan cyberpunk — lebih ke sci-fi game interface yang bersih

## 📝 Edit Konten

Semua data portfolio disimpan di `js/data.js` dalam bentuk array object:

- `skillMetrics` — Data skill bar
- `projects` — Data project card
- `skillCategories` — Data skill database
- `experiences` — Data timeline pengalaman
- `achievements` — Data achievement badge
- `typingRoles` — Role untuk animasi mengetik

Edit file tersebut untuk mengubah konten tanpa menyentuh HTML.
